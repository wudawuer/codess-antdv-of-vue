#### Form
Create a vertical Form
```
<Form>
    <FormItem label="Field A">
        <Input placeholder="input placeholder" />
    </FormItem>
    <FormItem label="Field B">
        <Input placeholder="input placeholder" />
    </FormItem>
    <FormItem>
        <Button type="primary" size="large">Submit</Button>
    </FormItem>
</Form>

```
